# Color Palette

**Hard rule: never use raw hex values. Always reference a token.**

## Primary
- Background: `#000000` — page background
- White: `#FFFFFF` — primary text on dark

## Accent
- Purple 500: `#a855f7` — primary CTAs, highlights, active states

## Surface
- surface.50: `#0a0a0a`
- surface.100: `#1a1a1a`
- surface.200: `#2a2a2a`
- surface.300: `#3a3a3a`
